[

    ],
    "tags": ["news", "premium_content"],

  {

  }

Required Fields
  • property_type: Type of property
  • name: Human-readable name
  • identifiers: Array of identifier objects
  • publisher_domain: Domain for verification

  • podcast_guid: guid-string
  • station_id: WXYZ-FM
